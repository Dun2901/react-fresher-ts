# 📚 Bookstore

Giới thiệu project: “Website bán sách”

Mục tiêu: bán sách online, quản lý đơn hàng, user, admin, tích hợp thanh toán.

## 🛠️ Tech Stack

**Client:**

- React 18 + Typescript
- React Router DOM v6
- Ant Design (antd) v5 + Ant Design Icons + Pro Components
- Axios
- React Icons
- React Spinners
- Sass

## Run Locally

Môi trường chạy dự án: Node.js v20+

Clone the project

```bash
# 1. Clone
git clone https://github.com/Dun2901/react-fresher-ts.git
**cd** react-fresher-ts

# 2. Install dependencies
npm install

# 3. Copy env
cp .env.example .env

# 4. Chạy dự án
npm run dev
```
